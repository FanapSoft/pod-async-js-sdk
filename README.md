## Synopsis

**Fanap's POD** chat service

## Code Example

```javascript
const Chat = require('podchat');

const options = {
	interval : 10000,
	retries : 10
};

Chat(options);

```

## Motivation

This module helps you to easily connect POD chat service.

## Installation

```javascript
npm install podchat --save
```
